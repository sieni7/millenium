## 📖 À propos
**KIRAM PHARMA** est une SARL basée à Abidjan (Cocody Angré Djibi), dédiée à l'excellence en santé. Dirigée par **Madame Fofana Geneviève**, l'entreprise réunit expertise locale et rigueur internationale pour la distribution et la promotion médicale.

## 🚀 Fonctionnalités "ProMax"
- 💎 **Interface Soft-UI** : Design épuré avec verre dépoli (glassmorphism) et ombres premium.
- 📱 **PWA Native-like** : Installation sur mobile, mode hors-ligne et icônes haute résolution.
- ⚡ **Performance Vite** : Chargement ultra-rapide grâce au bundling ES Modules.
- 🌓 **Mode Sombre** : Expérience utilisateur adaptée à toutes les conditions lumineuses.
- 🔍 **SEO & Indexation** : Score de **100/100** sur Lighthouse (JSON-LD, Sitemap, Robots.txt).
- ♿ **Inclusion Totale** : Entièrement navigable au clavier et compatible lecteurs d'écran.

## 💻 Installation & Usage

### Prérequis
- `Node.js` (v20+)
- `npm` ou `yarn`

### Commandes rapides
```bash
# Installation des dépendances
npm install

# Lancement en mode développement (Port 3000)
npm run dev

# Build pour la production
npm run build

# Aperçu local du build de production
npm run preview
```
